Research · Topological ML · HPC

NeuroTopo — Large-Scale Neuronal Morphology Analysis

End-to-end HPC pipeline processing ~286,000 reconstructed neurons (SWC format) to compute 8 topological Sholl descriptors and produce a 286,000×286,000 pairwise distance matrix (~164 GB, ~41 billion unique pairs) with O(1) lookup. Descriptor weights optimized via differential evolution over labeled neuronal classes. Published in PLOS Computational Biology Q1 (25 citations).

PythonNumPy / SciPyHDF5/h5py SLURMHPC A100TDANetworkX
Technical architecture

Pipeline stages

  • SWC parsing & standardization — soma centering, graph reordering, node role classification
  • 8 Sholl step-function descriptors: Branching Pattern, Tortuosity, Flux, Energy, Leaf, Wiring, Spread, TMD
  • Fixed-length vectorization: 8 × 50 = 400 float32 features per neuron stored in HDF5
  • Z-score normalization + PCA to 25D for clustering and classification
  • Block-wise upper-triangular distance matrix: 1,711 blocks at 5,000×5,000
  • Differential evolution weight optimizer: 92-worker deferred-update, permutation-tested p-values

Research · Deep Learning · Nature-target

NeuronAtlas — Universal Neuron Embedding Atlas

Self-supervised multimodal Transformer (12 layers, d=512, ~48.2M parameters) trained on 286,380 morphologically reconstructed neurons across 94 species. Learns a species-agnostic representation space enabling cross-species retrieval, morphological type discovery, and cell-type classification. Targeting Nature / Nature Methods publication.

PyTorchTransformersVICReg MultimodalAWS286K neurons · 94 species
Model & training details

Architecture

  • Pre-norm Transformer · 12 layers · 16 heads · d=512 · FFN 2048 · ~48.2M parameters
  • 3 input modalities: Morphometrics (21D) · Principal Vectors (100D) · Sholl Descriptors (400D) → 57 tokens
  • Multi-objective: VICReg (invariance/variance/covariance) + masked reconstruction + supervised CE (94 species)
  • 15-ablation systematic study; best ablation: supervised_only (morpho full)
  • Species Recall@1: 69.98% on held-out test; morphometrics dominate by ~11 percentage points

Architecture deep-dive

4-layer architecture

  • UI Layer: Dynamic input forms auto-generated from model configuration files — UI adapts automatically to each model's parameter schema
  • Application Layer: State management, input validation, event handling, results rendering (inventory.bundle.js)
  • Model Config Layer: All models externalized to CSV (solvers.csv, inputs.csv, outputs.csv) — new models added without modifying application code

eScriptorium HTR Pipeline — Historical Manuscript Digitization

End-to-end Handwritten Text Recognition pipeline to digitize the Dublin Wide Streets Commissioners archive — 17,500 pages of 18th-century handwritten manuscripts. Achieved 86.4% character accuracy (CER 0.136) on pilot. Produced gold-standard PAGE-XML training datasets establishing the foundation for the first fully searchable digital edition of the archive.
